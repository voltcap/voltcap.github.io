## Selected Projects

### Archaeology and Heritage

#### [Mesopotamian Archaeological Site Evaluation](https://github.com/voltcap/Mesopotamian_Archaeological_Site_Evaluation)
**MESOSAT** is a fault-tolerant ensemble model for archaeological site classification, combining three machine learning models, majority voting, fault-injection mechanisms, and evaluation visualisations such as confusion matrices and heatmaps.

#### [Mesopotamian Trade Network](https://github.com/voltcap/Mesopotamian_Trade_Network)
A digital humanities and archaeology project analysing distances between Mesopotamian city-states to identify central hubs within the trade network, including Eridu, Ur, Uruk, Nippur, Mari, Assur, Nineveh, Ebla, and Kultepe.

#### [Gaziantep Castle Sentinel-2 Satellite Visualiser](https://github.com/voltcap/Gaziantep-Castle-Sentinel2-Satellite-Visualiser)
A Google Earth Engine project for visualising Sentinel-2 and Google Satellite data over Gaziantep Castle for structural health monitoring and heritage observation.

#### [Gebze Pathfinder](https://github.com/voltcap/Gebze-Pathfinder)
A pathfinding and visualisation project using Dijkstra’s algorithm to identify an optimal route in the Gebze neighbourhood toward Osman Hamdi Bey’s house, combining computation with local historical geography.

### Other Projects

#### [Spotify Liked Songs Genre and Language Classifier](https://github.com/voltcap/Spotify-Liked-Songs-Genre-and-Language-Classifier)
An automated Spotify classifier and playlist creator that sorts liked songs by genre, sub-genre, and language.

#### [LevelUp](https://github.com/voltcap/LevelUpApp)
A quiz generation app that reads uploaded documents and generates multiple quiz formats using the Chimera model.
